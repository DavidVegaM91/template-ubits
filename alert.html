Componente Alert

Sistema de alertas con 4 variaciones usando tokens UBITS

Variaciones del Alert

Success Alert

Se archivó tu contenido de forma exitosa

Info Alert

Esta es una información importante que debes conocer

Warning Alert

Este es un mensaje de advertencia importante

Error Alert

Este es un mensaje de error que requiere atención

Demo Interactivo

Prueba crear alertas dinámicamente con diferentes opciones:

Ejemplos de Uso

HTML Básico

<div class="ubits-alert ubits-alert--success">
    <div class="ubits-alert-content">
        <div class="ubits-alert-icon">
            <i class="far fa-check-circle"></i>
        </div>
        <div class="ubits-alert-text ubits-body-sm-regular">
            Tu mensaje aquí
        </div>
    </div>
    <button class="ubits-alert-close">
        <i class="far fa-times"></i>
    </button>
</div>

JavaScript

// Crear alerta simple
UBITSAlertManager.success('Operación exitosa');

// Crear alerta con opciones
UBITSAlertManager.error('Error al guardar', {
    closable: true,
    duration: 5000,
    onClose: () => console.log('Alerta cerrada')
});

// Crear alerta en contenedor específico
UBITSAlertManager.createInContainer(
    document.getElementById('mi-contenedor'),
    {
        type: 'warning',
        message: 'Advertencia importante',
        closable: false
    }
);
