Prueba SubNav

Todas las variantes del componente SubNav

Template

Variante personalizable con 3 secciones de ejemplo. Ideal para crear nuevas implementaciones.

Uso: Esta variante es perfecta para crear nuevas implementaciones. Puedes personalizar los tabs según tus necesidades.

Aprendizaje

Navegación para módulos de aprendizaje con Inicio, Catálogo y Mis Cursos.

Uso: Ideal para plataformas de e-learning, cursos online y gestión de contenido educativo.

Desempeño

Navegación para módulos de evaluación y seguimiento de desempeño.

Uso: Perfecto para sistemas de evaluación de empleados, seguimiento de objetivos y reportes de desempeño.

Encuestas

Navegación simple para módulos de gestión de encuestas.

Uso: Ideal para sistemas de encuestas, formularios y recolección de datos.

Tareas

Navegación para módulos de gestión de tareas y proyectos.

Uso: Perfecto para sistemas de gestión de proyectos, planificación y seguimiento de tareas.

Documentación

Navegación especial para documentación con hamburger menu responsive.

Uso: Especial para documentación técnica con hamburger menu que se adapta a pantallas pequeñas.
