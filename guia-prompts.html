Guía de Prompts UBITS

Prompts para usar en Cursor AI. Sin conocimiento técnico necesario, solo copia, pega y personaliza.

Importante: Estos prompts están diseñados para Cursor AI.

Cómo usar esta guía

1
Copia el prompt
Haz clic en el botón "Copiar" del prompt que necesites. Se copiará automáticamente a tu portapapeles.
2
Personaliza el prompt
Reemplaza los textos entre corchetes [TEXTO] con tu información específica antes de usar el prompt.
3
Pega en Cursor AI
Abre Cursor AI, pega el prompt en el chat y presiona Enter. Cursor hará todo el trabajo por ti.
4
Revisa y ajusta
Revisa los cambios que hizo Cursor y pide ajustes si es necesario. ¡Es así de fácil!

1. Clonar el repositorio UBITS

Usa este prompt para obtener el template UBITS completo en tu computadora por primera vez.

Quiero trabajar contigo usando el UBITS Template que está en este repositorio: https://github.com/DavidVegaM91/template-ubits.git 1. Importa el repositorio. 2. Lee los archivos README.md y .cursor/rules/cursor-rules.mdc para entender el contexto del proyecto. 3. Confírmame que comprendiste la configuración y pregúntame qué página quiero editar o qué nuevo flujo quiero crear.

2. Crear página de inicio del proyecto

Usa este prompt para configurar la página principal de tu proyecto UBITS.

IMPORTANTE: Antes de enviar este prompt a Cursor, reemplaza todos los textos entre [corchetes] con la información específica de tu proyecto.
Necesito editar la página principal de mi proyecto UBITS. Por favor: 1. **CONFIGURACIÓN SIDEBAR, SUB-NAV Y TAB-BAR:** - Edita `index.html` - Quiero que el sidebar tenga activo: [opciones: aprendizaje, diagnóstico, desempeño, encuestas, reclutamiento, tareas, ubits-ai, ninguno] - Quiero que el sub-nav sea la variante de: [opciones: template, aprendizaje, desempeno, encuestas, tareas] - Para responsive, quiero que el tab-bar tenga activo el item de: [opciones: modulos, perfil] 2. **CONTENIDO PRINCIPAL:** - Reemplaza el content-wrapper con: [dile qué contenido quieres en tu página de inicio, dale todo el contexto que consideres necesario] 4. **VERIFICACIÓN:** - Confirma que todos los tokens ubits-colors estén aplicados correctamente y no haber usado colores hardcodeados - Si se usaron botones confirma que usaste los botones de ubits - **VALIDA que TODAS las clases de tipografía sean de la lista oficial de UBITS (ubits-heading-h1, ubits-heading-h2, ubits-body-md-regular, etc.) - NUNCA inventes clases como ubits-h1, ubits-h2, ubits-title, ubits-text, ubits-paragraph** - **ANTES de usar cualquier clase tipográfica, VERIFICA que esté en la lista oficial del UBITS-TYPOGRAPHY-GUIDE.md** - Procura usa componentes UBITS existentes o crea nuevos solo si lo necesitas - Asegúrate de que la interfaz sea responsive y funcione en modo claro/oscuro

4. Actualizar el repositorio UBITS

Usa este prompt cuando quieras obtener las últimas actualizaciones del template sin perder tus cambios personalizados.

Necesito actualizar mi proyecto UBITS con las últimas versiones del repositorio, pero sin perder mis cambios personalizados. Por favor: 1. Haz un backup de mis archivos personalizados (todos los que he modificado) 2. Actualiza el repositorio con: git pull origin main 3. Si hay conflictos, resuélvelos preservando mis cambios personalizados 4. Restaura mis archivos personalizados después de la actualización 5. Verifica que todo funcione correctamente Mi objetivo es tener las últimas mejoras del template UBITS manteniendo mis personalizaciones.
