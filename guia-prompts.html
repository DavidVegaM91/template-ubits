Guía de Prompts UBITS

Prompts para usar en Cursor AI. Sin conocimiento técnico necesario, solo copia, pega y personaliza.

Importante: Estos prompts están diseñados para Cursor AI.

1. Clonar el repositorio UBITS

Usa este prompt para obtener el template UBITS completo en tu computadora por primera vez.

Quiero trabajar contigo usando el UBITS Template que está en este repositorio: https://github.com/DavidVegaM91/template-ubits.git 1. Importa el repositorio. 2. Lee los archivos README.md y .cursorrules para entender el contexto del proyecto. 3. Confírmame que comprendiste la configuración y pregúntame qué página quiero editar o qué nuevo flujo quiero crear.

2. Crear página de inicio del proyecto

Usa este prompt para configurar la página principal de tu proyecto UBITS.

IMPORTANTE: Antes de enviar este prompt a Cursor, reemplaza todos los textos entre [corchetes] con la información específica de tu proyecto.
Necesito crear la página de inicio de mi proyecto UBITS. Por favor: 1. **CONTEXTO DEL PROYECTO:** - [Dale el contexto completo del proyecto a crear - puede ser directo en el chat o subir un archivo .txt y decirle que lo lea] 2. **CONFIGURACIÓN DE LA PÁGINA INICIAL:** - Edita `index.html` (esta será mi página principal) - Quiero que el sidebar tenga activo: [opciones: aprendizaje, diagnóstico, desempeño, encuestas, reclutamiento, tareas, ubits-ai, ninguno] - [Especifica qué variante del TopNav usar: template, aprendizaje, desempeno, encuestas, tareas, o crear una nueva] - [Para responsive, indica qué tab del tab-bar quieres activo: modulos, perfil, modo-oscuro] 3. **CONTENIDO PRINCIPAL:** - Reemplaza el content-wrapper con: [dile qué contenido quieres en tu página de inicio] - Usa componentes UBITS existentes o crea nuevos según necesites - Asegúrate de que sea responsive y funcione en modo claro/oscuro 4. **VERIFICACIÓN:** - Confirma que todos los tokens UBITS estén aplicados correctamente - Verifica que la navegación funcione correctamente - Prueba en diferentes tamaños de pantalla

Necesito crear una nueva página UBITS. Por favor: 1. Abre el archivo plantilla-ubits.html 2. Crea una copia con el nombre: [nombre-de-mi-pagina].html 3. Verifica que sea responsive y funcione en modo claro y oscuro Usa plantilla-ubits.html como base para mantener la consistencia del template. Título de la página: [TITULO_DE_LA_PAGINA] Contenido: [DESCRIBE_EL_CONTENIDO_QUE_QUIERES] Sigue la estructura de las páginas existentes como index.html o profile.html.

4. Personalizar colores UBITS

Usa este prompt para cambiar los colores del sistema UBITS manteniendo la consistencia del diseño.

Necesito personalizar los colores de mi proyecto UBITS. Por favor: 1. Identifica qué colores quiero cambiar (ej: color principal, secundario, etc.) 2. Modifica los tokens correspondientes en ubits-colors.css 3. Asegúrate de actualizar tanto el modo claro como el oscuro 4. Verifica que todos los componentes usen los nuevos colores 5. Mantén la consistencia visual en toda la aplicación Colores que quiero cambiar: - Color principal: [NUEVO_COLOR_PRINCIPAL] - Color secundario: [NUEVO_COLOR_SECUNDARIO] - Otros colores: [DESCRIBE_OTROS_COLORES] Asegúrate de que los nuevos colores tengan buen contraste y accesibilidad.

5. Actualizar el repositorio UBITS

Usa este prompt cuando quieras obtener las últimas actualizaciones del template sin perder tus cambios personalizados.

Necesito actualizar mi proyecto UBITS con las últimas versiones del repositorio, pero sin perder mis cambios personalizados. Por favor: 1. Haz un backup de mis archivos personalizados (todos los que he modificado) 2. Actualiza el repositorio con: git pull origin main 3. Si hay conflictos, resuélvelos preservando mis cambios personalizados 4. Restaura mis archivos personalizados después de la actualización 5. Verifica que todo funcione correctamente Mi objetivo es tener las últimas mejoras del template UBITS manteniendo mis personalizaciones.

Cómo usar esta guía

1
Copia el prompt
Haz clic en el botón "Copiar" del prompt que necesites. Se copiará automáticamente a tu portapapeles.
2
Personaliza el prompt
Reemplaza los textos entre corchetes [TEXTO] con tu información específica antes de usar el prompt.
3
Pega en Cursor AI
Abre Cursor AI, pega el prompt en el chat y presiona Enter. Cursor hará todo el trabajo por ti.
4
Revisa y ajusta
Revisa los cambios que hizo Cursor y pide ajustes si es necesario. ¡Es así de fácil!
