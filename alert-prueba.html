Variantes del TopNav - Preview

Todas las variantes disponibles del componente TopNav

1. Template (Plantilla)

Propósito: Variante base personalizable
Uso: Ideal para nuevos proyectos que necesitan personalización completa

2. Documentación

Propósito: Para páginas de documentación del template
Características: Título "DOCUMENTACIÓN", menú hamburguesa responsive

3. Aprendizaje

Propósito: Para plataformas de aprendizaje
Uso: Cursos, catálogos, gestión de aprendizaje

4. Desempeño

Propósito: Para sistemas de evaluación de desempeño
Uso: Evaluaciones 360, objetivos, métricas, reportes

5. Encuestas

Propósito: Para sistemas de encuestas
Uso: Creación y gestión de encuestas

6. Tareas

Propósito: Para sistemas de gestión de tareas
Uso: Planes, tareas, gestión de proyectos
