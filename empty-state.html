Empty State

Componente para mostrar estados vacíos con icono, título, descripción y dos botones de acción (secondary y primary). Útil para mostrar cuando no hay contenido disponible, resultados de búsqueda, o estados iniciales.

Características generales

  • Icono en contenedor: Icono FontAwesome dentro de un contenedor con fondo bg-2.
  • Título y descripción: Textos personalizables con tipografía UBITS.
  • Dos botones opcionales: Botón Secondary y Primary con todas las variantes y tamaños de UBITS Button.
  • Responsive: Se adapta automáticamente a diferentes tamaños de pantalla.
  • Tokens UBITS: Usa colores y tipografía oficiales del sistema de diseño.

Documentación técnica

Requisitos obligatorios

<!-- CSS Requerido -->
<link rel="stylesheet" href="components/empty-state.css">
<link rel="stylesheet" href="components/button.css"> <!-- Si se usa botón de acción -->
<link rel="stylesheet" href="fontawesome-icons.css"> <!-- Si se usa icono en botón -->
<link rel="stylesheet" href="ubits-colors.css">
<link rel="stylesheet" href="ubits-typography.css">

<!-- JavaScript Requerido -->
<script src="components/empty-state.js"></script>

Implementación básica

<div id="mi-empty-state-container"></div>
<script>
    document.addEventListener('DOMContentLoaded', function() {
        loadEmptyState('mi-empty-state-container', {
            icon: 'fa-grid',
            iconSize: 'lg', // 'sm', 'md', 'lg' (default: 'lg')
            title: 'No se encontraron resultados',
            description: 'Intenta ajustar los filtros de búsqueda para encontrar lo que buscas.'
        });
    });
</script>

Con botones (secondary y primary)

<div id="mi-empty-state-container"></div>
<script>
    document.addEventListener('DOMContentLoaded', function() {
        loadEmptyState('mi-empty-state-container', {
            icon: 'fa-grid',
            title: 'No se encontraron resultados',
            description: 'Intenta ajustar los filtros de búsqueda para encontrar lo que buscas.',
            buttons: {
                secondary: {
                    text: 'Cancelar',
                    icon: 'fa-times',
                    onClick: function() {
                        console.log('Cancelar');
                    }
                },
                primary: {
                    text: 'Buscar',
                    icon: 'fa-search',
                    onClick: function() {
                        console.log('Buscar');
                    }
                }
            }
        });
    });
</script>

Parámetros disponibles

Parámetro Tipo Requerido Descripción
containerId string Sí ID del contenedor donde se renderizará el empty state
options.icon string No Clase FontAwesome del icono (ej: 'fa-grid', default: 'fa-grid')
options.iconSize string No Tamaño del contenedor: 'sm' (28x28px, icono 12px), 'md' (32x32px, icono 16px), 'lg' (36x36px, icono 20px). Default: 'lg'
options.title string Sí Título del empty state
options.description string Sí Descripción del empty state
options.buttons object No Configuración de los botones (secondary: {text, icon, onClick}, primary: {text, icon, onClick})

Métodos disponibles

const emptyStateInstance = loadEmptyState('mi-container', {...});

// Actualizar configuración
emptyStateInstance.update({
    title: 'Nuevo título',
    description: 'Nueva descripción'
});

// Obtener configuración actual
const config = emptyStateInstance.getConfig();
