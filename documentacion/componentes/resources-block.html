Resources block

Panel del paso Recursos del LMS Creator: selector de ocho tipos o formularios (video, archivos, SCORM, embebido) con acción Cancelar. Referencia Figma Learn-Components: Resources block (248:6265).

Preview del componente

Elige la variante con el Input select oficial (createInput). Incluye resources-card.js antes de resources-block.js; tras pintar el preview, initResourcesBlockFields monta los campos sm alineados con los botones sm.

Características generales

Un bloque, muchas vistas
Quince variantes alineadas al lienzo Figma: selector, video, PDF, mp4, SCORM, embebido y bloqueo.
Página sin recurso
La variante default-error es el mismo selector que default con borde rojo: al avanzar dejando la página sin recurso, al abrirla se muestra así.
Tokens y botones UBITS
Superficie bg-1, cancelar con botón error secundario oficial.
Compone con Resources card
La variante default reutiliza las tarjetas del componente Resources card.

Prompts útiles

Copia y personaliza. Indica importar dropdown-menu.css, input.css, button.css, resources-card.css, resources-block.css, dropdown-menu.js e input.js (antes de resources-block.js), más resources-card.js.

Panel Recursos

En [pantalla] del LMS Creator, añade el bloque UBITS «Resources block» para el paso Recursos: primero el selector de tipos (variante default) y al elegir un tipo muestra la variante correspondiente (video-empty, pdf-empty, etc.). Importa dropdown-menu.js e input.js antes de resources-card.js y resources-block.js; CSS: dropdown-menu, input, resources-card, resources-block y button. Tras asignar innerHTML con resourcesBlockHtml({ variant: '[variante]' }), llama initResourcesBlockFields(contenedor) para montar los Input UBITS (tamaño sm) y selects oficiales.
