Badge Tag

Badge tipo pill con punto de color o icono a la izquierda. Distinto del Status Tag: usa normalmente un punto; en ocasiones puede llevar icono FontAwesome. Estilos: outlined (fondo blanco, borde) y filled (fondo de color, texto blanco).

Características

  • Indicador: por defecto un punto de color (span.ubits-badge-tag__indicator); opcionalmente un icono FontAwesome: añadir clase ubits-badge-tag--with-icon al badge y usar <i class="far fa-check-circle"> (solo clases FontAwesome en el <i>, sin clases propias).
  • Estilo: --outlined (fondo blanco, borde, texto oscuro) o --filled (fondo de color, texto blanco).
  • Colores: success, info, warning, error.
  • Tamaños: sm (28px), md (32px), lg (36px).
  • Renderizado: HTML directo. No requiere JavaScript.

Estructura HTML

Con punto (por defecto): 

<span class="ubits-badge-tag ubits-badge-tag--outlined ubits-badge-tag--success ubits-badge-tag--sm">
  <span class="ubits-badge-tag__indicator"></span>
  <span class="ubits-badge-tag__text">Badge text</span>
</span>

Con icono (añadir ubits-badge-tag--with-icon y usar <i> solo con clases FontAwesome, sin clase propia): 

<span class="ubits-badge-tag ubits-badge-tag--outlined ubits-badge-tag--success ubits-badge-tag--sm ubits-badge-tag--with-icon">
  <i class="far fa-check-circle"></i>
  <span class="ubits-badge-tag__text">Completado</span>
</span>

Clases 

/* Estilo */
.ubits-badge-tag--outlined   /* Fondo blanco, borde, texto oscuro */
.ubits-badge-tag--filled    /* Fondo de color, texto blanco */

/* Color */
.ubits-badge-tag--success   /* Verde */
.ubits-badge-tag--info      /* Azul */
.ubits-badge-tag--warning   /* Amarillo/Naranja */
.ubits-badge-tag--error     /* Rojo */

/* Tamaño */
.ubits-badge-tag--sm        /* 28px min-height */
.ubits-badge-tag--md        /* 32px */
.ubits-badge-tag--lg        /* 36px */

/* Indicador: icono en lugar de punto */
.ubits-badge-tag--with-icon

Requisitos

CSS: components/badge-tag.css, general-styles/ubits-colors.css. Si usas icono: general-styles/fontawesome-icons.css.
