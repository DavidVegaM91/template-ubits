Badge Tag

Etiqueta pequeña con punto de color o icono para mostrar estados (completado, en proceso, error). Se usa en listas y tarjetas.

Preview del componente

Un solo ejemplo en vivo: elige estilo, color y tamaño con los selectores; usa los interruptores para punto, icono o solo icono. En filled con icono, el icono usa el acento semántico. Nota: en icon-only el componente fija 28×28px (el tamaño sm/md/lg no agranda esa variante).

Estilo

Color

Tamaño

Activa solo si usas icono. Útil en listas compactas (p. ej. task-strip).

Características generales

Estilo
Outlined (borde) o Filled (fondo de color)
Indicador
Punto de color o icono a la izquierda
Colores
Success, Info, Warning, Error
Tamaños
Tres tamaños: sm, md y lg

Prompts útiles

Copia el prompt, reemplaza lo que va entre [corchetes] y pide a Cursor que lo aplique. Asegúrate de que importe badge-tag.css y alert.css para que no se vea roto.

Insertar Badge Tag

Inserta un Badge Tag en [ubicación] con texto "[texto]", color [success, info, warning o error], estilo [outlined o filled] y tamaño [sm, md o lg]. Debe importar badge-tag.css y alert.css.

Badge Tag con icono

Inserta un Badge Tag en [ubicación] con icono [nombre del icono], texto "[texto]" y color [success, info, warning o error]. Debe importar badge-tag.css y alert.css.

Cambiar tamaño del Badge Tag

Cambia el Badge Tag de [ubicación o descripción] para que use tamaño [sm, md o lg].

Estructura del componente 

<!-- Badge Tag con punto (por defecto) -->
<span class="ubits-badge-tag ubits-badge-tag--outlined ubits-badge-tag--success ubits-badge-tag--sm">
  <span class="ubits-badge-tag__indicator"></span>
  <span class="ubits-badge-tag__text">Badge text</span>
</span>

<!-- Badge Tag con icono (clase --with-icon, <i> solo con FontAwesome) -->
<span class="ubits-badge-tag ubits-badge-tag--outlined ubits-badge-tag--success ubits-badge-tag--sm ubits-badge-tag--with-icon">
  <i class="far fa-check-circle"></i>
  <span class="ubits-badge-tag__text">Completado</span>
</span>

Clases disponibles 

/* Estilo */
.ubits-badge-tag--outlined   /* Fondo blanco, borde, texto oscuro */
.ubits-badge-tag--filled    /* Fondo de color, texto blanco */

/* Color */
.ubits-badge-tag--success   /* Verde */
.ubits-badge-tag--info      /* Azul */
.ubits-badge-tag--warning   /* Amarillo/Naranja */
.ubits-badge-tag--error     /* Rojo */

/* Tamaño */
.ubits-badge-tag--sm        /* 28px min-height */
.ubits-badge-tag--md        /* 32px */
.ubits-badge-tag--lg        /* 36px */

/* Indicador: icono en lugar de punto (solo clases FontAwesome en el <i>) */
.ubits-badge-tag--with-icon

Cómo implementar

🔧 REQUISITOS OBLIGATORIOS:

CSS Requerido:
• components/badge-tag.css - Estilos del componente
• general-styles/ubits-colors.css - Tokens UBITS
• general-styles/fontawesome-icons.css - Solo si usas icono

JavaScript: NO requerido - Componente puramente CSS

Renderizado: HTML directo con clases UBITS. Con icono: usar solo clases FontAwesome en el <i> (ej. far fa-check-circle), sin clases propias. 
<!-- Con punto -->
<span class="ubits-badge-tag ubits-badge-tag--outlined ubits-badge-tag--success ubits-badge-tag--sm">

</span>
