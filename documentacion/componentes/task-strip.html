Task strip

Fila compacta para una tarea en listas de operaciones: completado, nombre, estado, vencimiento, prioridad, persona asignada y acciones. Se usa en Tareas y en el detalle de un plan.

Preview del componente

El HTML lo genera renderTaskStrip(tarea, opts). Cambia el ejemplo con los botones; el interruptor oculta el botón «Agregar a un plan» (útil en vistas donde no aplica).

Características generales

Una fila, toda la información
Checkbox, título editable, etiqueta opcional y columna de acciones alineada
Variante aprendizaje
Icono de formación junto al título cuando la tarea es de tipo aprendizaje
Estado y prioridad UBITS
Status tag y badge de prioridad coherentes con el sistema de diseño
Asignación visible
Avatar, iniciales o placeholder según los datos de la tarea
Renderizado por función
Misma API en la lista de tareas y en el detalle de plan

Prompts útiles

Copia el prompt y personaliza lo que va entre corchetes. Indica siempre los CSS y JS del componente y de checkbox, button, status-tag, badge-tag y tooltip para que la tirilla se vea completa.

Listar tareas como tirillas

En [ubicación], renderiza una lista de tareas usando la función renderTaskStrip con los datos de [origen de datos]. Cada objeto debe tener al menos nombre, estado, fecha de vencimiento, prioridad y asignado. Importa task-strip.css, checkbox.css, button.css, status-tag.css, badge-tag.css, tooltip.js y task-strip.js, y llama a initTooltip en los elementos con data-tooltip después de pintar.

Tirilla de tarea de aprendizaje

Para las tareas de [lista o flujo] que sean de formación, marca cada ítem con taskType aprendizaje para que la tirilla muestre el icono de birrete junto al título. Mantén task-strip.css y el resto de dependencias del componente.

Sin botón agregar a plan

Al renderizar tirillas en [pantalla], pasa hideAddToPlan true en las opciones de renderTaskStrip para ocultar el botón de agregar a un plan.

Estructura generada (resumen)

Raíz .tarea-item con data-tarea-id. Modificadores: .tarea-item--completed, .tarea-item--overdue, .tarea-item--aprendizaje. 

<div class="tarea-item" data-tarea-id="...">
  <div class="tarea-item__main">
    <span class="tarea-item__radio">... ubits-checkbox redondo ...</span>
    <div class="tarea-content">
      <div class="tarea-titulo-wrap">... título + botón editar nombre ...</div>
      <!-- opcional: .tarea-etiqueta -->
    </div>
  </div>
  <div class="tarea-item__actions">
    <!-- status-tag, fecha, prioridad badge-tag, asignado, menú, detalle -->
  </div>
</div>

renderTaskStrip(tarea, opts)

Devuelve un string HTML. Tras insertarlo en el DOM, conviene llamar initTooltip en los nodos con data-tooltip.

Campo (tarea) Notas
id, name Identificador y título mostrado
done, status Finalizado y texto de estado interno (Activo, Finalizado, etc.)
endDate String comparable con opts.today (p. ej. YYYY-MM-DD) para vencimiento
priority alta | media | baja (por defecto media)
etiqueta Texto opcional bajo el bloque principal
taskType Si vale aprendizaje, layout con icono de birrete
assignee_* Nombre, correo o URL de avatar; o usar getAssignee / renderAvatar en opts
Opción (opts) Notas
today Fecha de referencia para marcar vencidas
esVencidaSection Si es true, la fila se trata como vencida aunque la fecha no lo indique (bloque de vencidas)
formatDate, escapeHtml Formateo de fecha en pantalla y escape de texto (por defecto básico)
getAssignee, renderAvatar Personalizar quién se muestra y cómo (p. ej. componente Avatar)
hideAddToPlan Oculta el botón de agregar a plan

Edición inline del nombre

startInlineEditTaskName(tareaItemElement, taskId, onSave) sustituye el título por un input; al guardar se invoca onSave(nuevoNombre). Debes enlazar el gesto (doble clic o botón lápiz) en tu página y persistir el cambio donde corresponda.

Ejemplo mínimo

var tarea = {
  id: 1,
  name: 'Mi tarea',
  done: false,
  status: 'Activo',
  endDate: '2026-04-10',
  priority: 'media',
  assignee_name: 'Ana'
};
var html = renderTaskStrip(tarea, {
  today: '2026-03-31',
  formatDate: function (d) {
    if (!d) return '';
    var p = d.split('-');
    return p[2] + '/' + p[1] + '/' + p[0];
  }
});
document.getElementById('lista').innerHTML = html;
initTooltip('#lista [data-tooltip]');
