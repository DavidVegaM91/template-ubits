Dropdown Menu

Menú desplegable unificado: padding sm, fondo blanco (bg-1), elevation, texto regular alineado a la izquierda. Opcional: autocomplete (p. ej. encabezados de tabla), opciones con icono/checkbox/switch a izquierda o derecha. Footer con botón secundario y primario a la derecha.

Preview del componente

Haz clic en "Abrir menú" para ver el resultado. Los controles debajo permiten activar/desactivar autocomplete y footer, y elegir una variante para las opciones. Hover usa bg-2.

Características

Estilo unificado
Padding sm, fondo bg-1, elevation, texto regular alineado a la izquierda
Input de búsqueda opcional
Lupa y botón limpiar; para filtrar opciones (p. ej. encabezados de tabla)
Opciones flexibles
Icono, checkbox o switch a izquierda o derecha de cada ítem
Footer
Dos botones a la derecha: secundario y primario

Prompts útiles

Usa estos prompts con Cursor AI para implementar el Dropdown Menu.

Usar Dropdown Menu UBITS

Usa el componente Dropdown Menu UBITS para los menús desplegables. Importa components/dropdown-menu.css y components/dropdown-menu.js. Genera el HTML con getDropdownMenuHtml({ overlayId, options, hasAutocomplete?, autocompletePlaceholder?, footerSecondaryLabel?, footerPrimaryLabel? }). Posiciona y abre con openDropdownMenu(overlayId, anchorElement). Cierra con closeDropdownMenu(overlayId) o clic en el overlay. Opciones: array de { text, value?, leftIcon?, rightIcon?, checkbox?, switch?, selected? }.

Reemplazar menús por Dropdown Menu

Reemplaza los menús desplegables (prioridad, estado, filtros, ordenamiento) por el componente Dropdown Menu UBITS. Usa getDropdownMenuHtml() con las mismas opciones y footer (Cancelar / Aplicar donde aplique). Mantén la lógica de open/close con openDropdownMenu(overlayId, anchorElement) y closeDropdownMenu(overlayId). Importa components/dropdown-menu.css y components/dropdown-menu.js.

Estructura, clases y API del componente Dropdown Menu.

Estructura

.ubits-dropdown-menu__overlay   /* Overlay transparente, cierre al clic */
.ubits-dropdown-menu__content   /* Panel: padding sm, bg-1, elevation, border-radius-sm */
  .ubits-dropdown-menu__autocomplete-wrap  /* Opcional: contenedor para input autocomplete */
  .ubits-dropdown-menu__options
    .ubits-dropdown-menu__option         /* Botón por ítem: left + text + right */
      .ubits-dropdown-menu__option-left  /* Icono o checkbox */
      .ubits-dropdown-menu__option-text  /* Texto regular, alineado izquierda */
      .ubits-dropdown-menu__option-right /* Icono o switch */
  .ubits-dropdown-menu__footer           /* Secundario + Primario a la derecha */

API JavaScript

getDropdownMenuHtml(config)
  config.overlayId, config.contentId
  config.options = [{ text, value?, leftIcon?, rightIcon?, checkbox?, switch?, selected? }]
  config.hasAutocomplete?, config.autocompletePlaceholder?, config.autocompleteContainerId?
  config.footerSecondaryLabel?, config.footerPrimaryLabel?
  config.footerSecondaryId?, config.footerPrimaryId?

openDropdownMenu(overlayId, position)
  position = { top, left } en px, o elemento ancla (getBoundingClientRect)

closeDropdownMenu(overlayId)
