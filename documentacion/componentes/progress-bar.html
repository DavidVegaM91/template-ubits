Progress bar

Barra horizontal para mostrar avance de un contenido, plan o carga (0–100 %). Unifica los grosores que hoy aparecen en cards de aprendizaje, tablas, planes, file upload y zona de estudio del home.

Por defecto el relleno es azul marca (--ubits-accent-brand). La variante chart (gris-azul de gráficas) solo para escenarios puntuales y poco comunes; no usarla en Card Content ni en flujos habituales de producto.

Preview del componente

Dos alturas oficiales: sm (5px) para cards, overlays, upload y zona de estudio; lg (8px) para tablas y planes. Usa el interruptor para alternar bordes rectos o tipo pill, y los botones de valor para ver el relleno y el estado completado (100 %).

Características generales

Dos alturas
sm 5px (cards, upload, home) y lg 8px (tablas, planes). xs y md se normalizan a sm.
Pill opcional
Con bordes redondeados tipo pastilla o esquinas rectas para overlays de card.
Completado
Al llegar a 100 % el relleno pasa a verde de éxito (o puedes forzarlo con status).
Azul por defecto
Relleno accent-brand en cards, tablas, planes y home. variant: 'chart' solo excepciones.
HTML o función
Genera markup con progressBarHtml o monta con createProgressBar en un contenedor.

Prompts útiles

Insertar barra de progreso

Inserta una Progress bar UBITS en [ubicación] con valor [porcentaje], tamaño [sm/lg] y bordes redondeados [sí/no]. Importa progress-bar.css y progress-bar.js.

Actualizar avance

Cuando cambie el progreso de [elemento], actualiza la barra con setValue del objeto devuelto por createProgressBar o con setProgressBarValue sobre el nodo .ubits-progress-bar.

Estructura HTML

<div class="ubits-progress-bar ubits-progress-bar--sm ubits-progress-bar--rounded"
     role="progressbar" aria-valuemin="0" aria-valuemax="100" aria-valuenow="65"
     aria-label="Progreso del plan">
  <div class="ubits-progress-bar__track">
    <div class="ubits-progress-bar__fill" style="width:65%"></div>
  </div>
</div>

createProgressBar / progressBarHtml

var bar = createProgressBar({
  containerId: 'mi-barra',
  value: 65,
  size: 'sm',        // sm | lg
  rounded: true,     // pill; false = esquinas rectas
  variant: 'default', // omitir variant = azul marca; 'chart' solo excepciones puntuales
  track: 'default',  // 'static' | 'subtle'
  width: '100%',     // opcional, ej. '60px' en tablas
  withLabelRow: false, // true → progressBarRowHtml + %
  fixedWidth60: false,
  ariaLabel: 'Progreso'
});
// Tabla: tableProgressCellHtml(75) → fila lg 60px + %
