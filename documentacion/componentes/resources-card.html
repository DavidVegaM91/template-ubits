Resources card

Tarjeta compacta para elegir el tipo de recurso en el paso Recursos del LMS Creator. Referencia Figma Learn-Components: Resources card (247:5165).

Preview del componente

Cambia el estado y el tipo. «Hover forzado» aplica la clase is-hover (mismo borde que al pasar el ratón). «Deshabilitado» usa el atributo disabled del botón.

Características generales

Tokens UBITS
Fondos, bordes, marca y texto deshabilitado alineados con el sistema de color.
Medidas Figma
Ancho máximo 153,75px, alto fijo 86px, padding vertical y horizontal con tokens, radio grande; etiqueta hasta 2 líneas con puntos suspensivos.
Doce tipos
Icono y etiqueta por tipo de recurso del flujo de creación.

Prompts útiles

Copia y personaliza. Indica importar resources-card.css, ubits-colors.css, ubits-typography.css, fontawesome-icons.css y resources-card.js.

Selector de tipo de recurso

En el panel derecho del paso Recursos en [nombre de pantalla], añade una cuadrícula de tarjetas con el componente UBITS «Resources card» para elegir entre video, PDF, texto y el resto de tipos. Importa resources-card.css y resources-card.js; usa tokens UBITS y la función resourcesCardHtml({ type: '[tipo]' }).

Estructura HTML

<button type="button" class="ubits-resources-card" data-resources-card-type="video" aria-label="Video">
  <span class="ubits-resources-card__inner">
    <span class="ubits-resources-card__icon-wrap" aria-hidden="true"><i class="far fa-video"></i></span>
    <span class="ubits-resources-card__label">Video</span>
  </span>
</button>

Clases y estados

Raíz: .ubits-resources-card (<button>; data-resources-card-type)
  :hover / .is-hover (sin disabled) → borde 2px --ubits-accent-brand
  :disabled → fondo --ubits-bg-disabled, texto e icono --ubits-fg-disabled
  :focus-visible → anillo foco accesible
Hijos: .ubits-resources-card__inner, __icon-wrap (> i.far.fa-*), __label (12px/18px; máx. 2 líneas + ellipsis; fg medio)
Grid doc: .ubits-resources-card-grid

Requisitos y API

CSS: ubits-colors.css, ubits-typography.css, fontawesome-icons.css, resources-card.css
JS: resources-card.js
  resourcesCardHtml({ type, disabled?, className?, hoverClass?, ariaLabel? })
  RESOURCES_CARD_TYPES_ORDER — orden de columnas Figma
  RESOURCES_CARD_META — icono (clase FA sin prefijo) y etiqueta por type
Tipos type: video, video-desktop, pdf, texto, embebido, scorm, evaluacion-final,
  encuesta-libre, encuesta, archivo-descargable, certificado, imagen
