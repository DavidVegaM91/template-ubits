Tabla

Tabla para listar datos en filas y columnas. Permite texto, checkboxes, avatares, barras de avance, etiquetas de estado y prioridad. Se adapta a pantallas pequeñas con scroll horizontal.

Características generales

Columnas
Puedes añadir o quitar columnas según lo que necesites mostrar
Tipos de celda
Texto, checkbox, barra de avance, avatar, etiqueta de estado, prioridad
Móvil
En pantallas pequeñas la tabla hace scroll horizontal
Encabezados
Filtros, orden o meta fija (peso %) junto al título truncable

Encabezado con meta

Cuando el título de una columna es largo y lleva un dato fijo (p. ej. peso %), el patrón es: título truncable + texto secundario a la derecha siempre visible. Activa el checkbox del preview para verlo.

Botones en encabezados de columna

Las columnas pueden incluir botones de acción en el th. Según el tipo de acción, se debe usar un icono u otro para que la persona entienda qué opciones va a encontrar al hacer clic:

Filtros
Cuando el botón abre opciones de filtrado (por valor, checkboxes, etc.), usar el icono fa-filter.
Organización
Cuando el botón abre un menú con varias opciones de organización (no solo orden), usar el icono fa-ellipsis (tres puntos).
Orden
Cuando el botón solo abre opciones de orden (más reciente primero / al final, etc.), usar el icono fa-arrow-up-arrow-down. Es más intuitivo que los tres puntos para este caso.

Nota: Un mismo encabezado puede tener hasta 2 botones de acción (por ejemplo, filtro + orden). No está limitado a un solo botón por columna.

Prompts útiles

Copia el prompt, reemplaza lo que va entre [corchetes] y pide a Cursor que lo aplique. Asegúrate de que importe table.css y, si usas status tag o avatar, status-tag.css y avatar.css; alert.css para que no se vea roto.

Insertar Tabla

Inserta una tabla UBITS en [ubicación] con columnas [lista de columnas] y tipos de celda que necesites (texto, checkbox, avatar, barra de avance, etiqueta de estado, prioridad). Debe importar table.css, y si usas status tag o avatar: status-tag.css, avatar.css y alert.css.

Añadir columna

Añade a la tabla de [ubicación] una columna "[nombre]" con tipo [texto, checkbox, avatar, avance, estado o prioridad]. Debe importar table.css y alert.css.
