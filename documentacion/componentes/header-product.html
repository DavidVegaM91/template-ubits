Header Product

Componente de encabezado para páginas de producto. Incluye botón de retroceso, título del producto, botones de acción y breadcrumb opcionales.

Características generales

Componente flexible
Todos los elementos son opcionales y configurables
Responsive
Se adapta automáticamente a diferentes tamaños de pantalla
Abraza su contenido
Se ajusta automáticamente sin alturas fijas
Breadcrumb opcional
Configurable con múltiples niveles (máx. 3 recomendado)

Implementación básica 

<!-- Contenedor para el header-product -->
<div id="header-product-container"></div>

<!-- JavaScript para cargar el componente -->
<script src="../../components/header-product.js"></script>
<script>
    loadHeaderProduct('header-product-container', {
        productName: 'Name product',
        breadcrumbItems: [
            { text: 'Default', active: false },
            { text: 'Default', active: false },
            { text: 'Active', active: true }
        ],
        backButton: {
            onClick: function() {
                console.log('Back button clicked');
            }
        },
        infoButton: {
            onClick: function() {
                console.log('Info button clicked');
            }
        },
        aiButton: {
            text: 'AI button',
            onClick: function() {
                console.log('AI button clicked');
            }
        },
        secondaryButtons: [
            { text: 'Button text', icon: 'fa-th', onClick: function() { console.log('Secondary button clicked'); } }
        ],
        primaryButton: {
            text: 'Primary action',
            icon: 'fa-th',
            onClick: function() {
                console.log('Primary button clicked');
            }
        },
        menuButton: {
            onClick: function() {
                console.log('Menu button clicked');
            }
        }
    });
</script>
