Color picker

Selector de color en panel flotante para elegir un color con precisión sin salir del contexto. Sirve junto a un disparador cuadrado (como en el modal SCORM del LMS Creator): el usuario ajusta saturación y brillo, el matiz y el código hexadecimal, y el color se aplica al elemento anclado al cerrar o mientras arrastra.

Preview del componente

Pulsa el cuadrado de color para abrir el panel. Puedes fijar un color de partida con los botones de muestra antes de abrir. Escape o clic fuera cierran el panel.

Cómo implementar

Obligatorio: CSS y JS

Obligatorio: color-picker.css, input.css y input.js (antes de color-picker.js) para el campo HEX vía createInput. Cuentagotas si EyeDropper: button.css y fontawesome-icons.css. Solo un panel activo a la vez.

Referencia en producto

LMS Creator: modal recurso SCORM (scorm-recurso-modal.js, fila de color del embebido). La página crear-contenido.html ya enlaza input.css, input.js, color-picker.css y color-picker.js (orden correcto respecto al modal).

Parámetros de openColorPickerPopover

  • anchorEl (HTMLElement): elemento de referencia para posicionar y para pintar background con el HEX en vivo.
  • initialHex (string, opcional): formato #RRGGBB; si no es válido se usa azul marca por defecto.
  • onChange (función opcional): recibe el HEX en mayúsculas cada vez que cambia el color.
  • onClose (función opcional): al cerrar por Escape, clic fuera o close().
  • zIndex (número opcional): por defecto el CSS del panel aplica 2000; puedes subirlo si queda bajo otros overlays.

Retorno: null si falta ancla; si no, { close, getHex, setHex }. closeColorPickerPopover() cierra el singleton sin ancla.

Notas

  • Los gradientes del lienzo SV y del matiz usan RGB estándar del modelo HSV, no tokens de marca.
  • Cuentagotas: solo si typeof EyeDropper === "function" (Chrome, Edge, Safari reciente). Al muestrear se quita temporalmente el cierre por clic fuera para no cerrar el panel antes de elegir el píxel. Firefox u otros sin API no muestran el botón.
  • Clic fuera no cierra si el objetivo es el propio anchorEl (evita cierre inmediato en algunos flujos).
  • El HEX se monta con createInput (labelPosition: 'left', tamaño sm).

<script>
  var sw = document.getElementById('mi-swatch');
  var colorHex = '#0C5BEF';
  sw.style.width = 'var(--size-4xl)';
  sw.style.height = 'var(--size-4xl)';
  sw.style.borderRadius = 'var(--border-radius-sm)';
  sw.style.background = colorHex;
  sw.addEventListener('click', function () {
    openColorPickerPopover({
      anchorEl: sw,
      initialHex: colorHex,
      onChange: function (hex) {
        colorHex = hex;
        var out = document.getElementById('hex-out');
        if (out) out.textContent = hex;
      }
    });
  });
</script>
