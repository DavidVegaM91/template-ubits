Color picker

Selector de color en panel flotante para elegir un color con precisión sin salir del contexto. Sirve junto a un disparador cuadrado (como en el modal SCORM del LMS Creator): el usuario ajusta saturación y brillo, el matiz y el código hexadecimal, y el color se aplica al elemento anclado al cerrar o mientras arrastra.

Preview del componente

Características generales

Área saturación / brillo
Lona interactiva con cursor circular para afinar el color sobre el matiz elegido.
Franja de matiz
Recorre el arco iris de tonos y actualiza la vista previa y el valor hexadecimal.
Entrada HEX
Campo de texto oficial UBITS (createInput): código con almohadilla y seis hexadecimales para fijar el color.
Anclaje flotante
El panel se posiciona junto al disparador y puede pintar su fondo al elegir color, como un muestrario.
