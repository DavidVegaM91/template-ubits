Coachmark

Product tour: máscara, spotlight en el elemento y texto del paso con el mismo Popover UBITS. Pasos, navegación y Escape para salir.

Preview del componente

Pulsa Iniciar tour de ejemplo para recorrer cuatro pasos: tres anclados a botones y uno sin ancla (solo scrim + popover centrado).

Características generales

Popover por paso
Mismo componente Popover: colita, alineación y botones UBITS
Spotlight
Hueco sobre el ancla; sin ancla, scrim a pantalla completa
Navegación
Flechas anterior/siguiente, check para finalizar, cierre en cabecera; bloqueo de clics fuera del panel

Capas

#ubits-coachmark-root (z-index configurable) contiene scrim y spotlight. El Popover del paso se monta en document.body con z-index por encima (p. ej. 10060).

API

UBITS_COACHMARK.start(options) — steps (array de target selector/elemento/función o null, title, body o bodyHtml, placement, align, offset, size, showSpotlight, requireTarget), popoverId, startIndex (índice 0-based del primer paso; útil en tours multi-página), onBeforeNext(index, step) / onBeforePrev(index, step) (devolver false para cancelar el paso y p. ej. navegar con location.href), showProgress (si es true y hay más de un paso: texto N de M en el pie izquierdo; cabecera con título + botón cierre icon-only (fa-times, como en modal) con tooltip «Cerrar» a la derecha salvo en el último paso, donde solo el botón de finalizar icon-only con fa-check), padding (default 0, exacto al target), zIndexLayer, zIndexPopover, onComplete (solo al activar finalizar), onDismiss(reason) (close | escape | missing-target | dismiss; no se llama con restart), onStepChange(index, step).

UBITS_COACHMARK.close(reason) — cierra tour y popover.
UBITS_COACHMARK.next() / prev() — avanza o retrocede paso.
UBITS_COACHMARK.layout() — recalcula spotlight y posición del popover (tras cambios de layout).
UBITS_COACHMARK.isActive() — boolean.

Ejemplo mínimo 
UBITS_COACHMARK.start({
  popoverId: 'mi-tour',
  showProgress: true,
  steps: [
    { target: '#paso-1', title: 'Bienvenida', body: 'Texto del paso.', placement: 'bottom', align: 'center' },
    { target: null, title: 'Resumen', body: 'Sin ancla: scrim + popover centrado.' }
  ],
  onComplete: function () { /* tour terminado con Finalizar */ },
  onDismiss: function (reason) { /* close | escape | missing-target | end */ }
});

Troubleshooting técnico

Bugs comunes y cómo resolverlos 
1) El tour no abre:
- Verifica que popover.js cargue antes de coachmark.js.
- UBITS_COACHMARK.start devuelve false si faltan pasos o openPopover no existe.

2) El spotlight no coincide con el target:
- Por defecto padding=0 (exacto al elemento).
- Si quieres margen, define padding: 4 o padding: 8 en start().

3) Tooltip de la X no aparece:
- Importa tooltip.css y tooltip.js.
- El botón de cierre usa data-tooltip y se inicializa con initTooltip.

4) El tooltip queda debajo del popover:
- Mantén #ubits-tooltip-container por encima del z-index del popover.
- En coachmark.css se fuerza z-index alto mientras el tour está activo.

5) Navegación multi-página:
- Usa startIndex para continuar en otro paso.
- Usa onBeforeNext/onBeforePrev y retorna false si vas a redirigir con location.href.
