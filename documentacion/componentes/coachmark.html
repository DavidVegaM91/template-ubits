Coachmark

Product tour: máscara, spotlight en el elemento y texto del paso con el mismo Popover UBITS. Pasos, navegación y Escape para salir.

Spotlight

Navegación

Documentación técnica

API JavaScript, capas DOM y dependencias.

Mostrar

Requisitos

  • components/coachmark.css + components/coachmark.js
  • components/popover.css + components/popover.js
  • components/button.css + fontawesome-icons.css
  • Estilos base: ubits-colors.css, ubits-typography.css, styles.css

Capas

#ubits-coachmark-root (z-index configurable) contiene scrim y spotlight. El Popover del paso se monta en document.body con z-index por encima (p. ej. 10060).

API

UBITS_COACHMARK.start(options) — steps (array de target selector/elemento/función o null, title, body o bodyHtml, placement, align, offset, size, showSpotlight, requireTarget), popoverId, startIndex (índice 0-based del primer paso; útil en tours multi-página), onBeforeNext(index, step) / onBeforePrev(index, step) (devolver false para cancelar el paso y p. ej. navegar con location.href), showProgress, padding, zIndexLayer, zIndexPopover, onComplete (solo al pulsar Finalizar), onDismiss(reason) (close | escape | missing-target | dismiss; no se llama con restart), onStepChange(index, step).

UBITS_COACHMARK.close(reason) — cierra tour y popover.
UBITS_COACHMARK.next() / prev() — avanza o retrocede paso.
UBITS_COACHMARK.layout() — recalcula spotlight y posición del popover (tras cambios de layout).
UBITS_COACHMARK.isActive() — boolean.

Ejemplo mínimo 
UBITS_COACHMARK.start({
  popoverId: 'mi-tour',
  showProgress: true,
  steps: [
    { target: '#paso-1', title: 'Bienvenida', body: 'Texto del paso.', placement: 'bottom', align: 'center' },
    { target: null, title: 'Resumen', body: 'Sin ancla: scrim + popover centrado.' }
  ],
  onComplete: function () { /* tour terminado con Finalizar */ },
  onDismiss: function (reason) { /* close | escape | missing-target | end */ }
});
