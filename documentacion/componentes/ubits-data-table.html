Data Table

Tabla de datos configurable para listar registros con búsqueda, filtros, orden, selección y acciones. Ideal para listas de tareas, colaboradores, planes o cualquier contenido tabular. Se configura con un objeto de opciones y una función que construye cada fila.

Preview del componente

Tabla generada con createUbitsDataTable(options): título, búsqueda, columnas visibles, botón primario, checkboxes, orden y filtros por columna.

Características generales

Columnas configurables
Define columnas con etiqueta, orden y filtro por columna.
Checkboxes y selección
Selección por fila y "seleccionar todo" con estado indeterminado.
Búsqueda y filtros
Búsqueda global y filtros por columna con chips y limpiar.
Ver seleccionados
Modo para mostrar solo las filas seleccionadas; el botón muestra el contador entre paréntesis (p. ej. Ver seleccionados (3)).
Columnas visibles
Dropdown para mostrar u ocultar columnas sin recargar.
Botón primario
Botón con texto e icono opcional en el header (p. ej. "Nueva tarea").

Checklist para tabla nueva

Pasos reutilizables para añadir una nueva tabla con el componente.

  • Incluir ubits-data-table.css y ubits-data-table.js en la página (y si usas búsqueda/filtros/columnas: table.css, button.css, input.css, checkbox.css, dropdown-menu.css, tooltip.css según features).
  • Definir el config: containerId, columns, getData, rowIdField, buildRowHtml, features (checkboxes, search, filters, verSeleccionados, actionBar, resultsCount, columnsToggle), y opcionalmente primaryButton, emptyState, emptySearchState, actionBarButtons, i18n, title.
  • Implementar buildRowHtml(row) para devolver el HTML de las celdas (sin la columna de checkbox; el componente la añade si features.checkboxes es true). Usar data-col="idColumna" en cada td para que orden y filtros funcionen.
  • Probar: checkboxes (seleccionar todo / fila), empty state sin datos, búsqueda, filtros por columna, ver seleccionados, barra de acciones y botón primario si los activaste.

Añadir tabla de datos en una página

Añade una tabla de datos UBITS en [ubicación de la página] usando createUbitsDataTable. Incluye ubits-data-table.css y ubits-data-table.js. Config: containerId [id del contenedor], columns [lista de columnas con id y label], getData que devuelva los datos, rowIdField "id", buildRowHtml que devuelva el HTML de las celdas con data-col por columna. Activa features: checkboxes, search, resultsCount, columnsToggle. Añade emptyState con mensaje [mensaje]. Si quieres un botón principal en el header, usa primaryButton: { text: "[texto]", icon: "fa-plus", onClick: function() {} }.

Opciones del config, API y requisitos.

Mostrar

Opciones del config (createUbitsDataTable)

  • containerId (string): ID del elemento donde se renderiza la tabla.
  • tableId (string, opcional): id del <table>. Por defecto "ubits-data-table".
  • columns (array): { id, label, sortable?, filterable? }.
  • getData (function): devuelve array de filas.
  • rowIdField (string): clave del objeto fila para el id (ej. "id").
  • buildRowHtml (function(row)): devuelve HTML de las celdas (sin checkbox). Cada td debe tener data-col="idColumna".
  • features (object): checkboxes, search, filters, verSeleccionados, actionBar, resultsCount, columnsToggle (boolean).
  • emptyState (object): message, icon. Estado cuando no hay datos.
  • emptySearchState (object): message, description, buttonText. Estado cuando la búsqueda no devuelve resultados.
  • actionBarButtons (array): { id, label, icon, onClick(selectedIds), variant? }.
  • primaryButton (object): { text, icon?, onClick() }. Botón primario en el header a la derecha del botón Columnas.
  • i18n (object): selectAll, deselectAll, verSeleccionados, clearFilters, filtrosAplicados, buscar, columnasVisibles.
  • title (string): título opcional en la barra superior.

Ejemplo mínimo

createUbitsDataTable({
  containerId: 'mi-tabla-container',
  columns: [ { id: 'nombre', label: 'Nombre', sortable: true }, { id: 'estado', label: 'Estado', filterable: true } ],
  getData: function() { return [ { id: '1', nombre: 'Ejemplo', estado: 'Activo' } ]; },
  rowIdField: 'id',
  buildRowHtml: function(row) {
    return '<td data-col="nombre"><span class="ubits-body-sm-regular">' + (row.nombre || '') + '</span></td><td data-col="estado"><span class="ubits-body-sm-regular">' + (row.estado || '') + '</span></td>';
  },
  features: { checkboxes: true, search: true, resultsCount: true }
});
