Date picker modal

Modal tamaño SM con calendario oficial UBITS (createCalendar). Soporta fecha única o rango (inicio y fin). Los campos aceptan escritura manual en formato DD/MM/YYYY. Referencia de uso: filtro «Personalizado» y cambio de fecha en seguimiento.html.

Preview del componente

Abre cada variante. Cierra con X, Cancelar, Aplicar o clic fuera del contenido.

Características generales

Modal SM
Usa getModalHtml + clases date-picker-modal-*
createCalendar
Modo simple o range: true según mode
Entrada manual
Campos DD/MM/YYYY sincronizados con el calendario

Requisitos (orden)

CSS: modal.css, button.css, input.css, calendar.css, date-picker-modal.css

JS: input.js → calendar.js → modal.js → date-picker-modal.js

Uso básico

var picker = createDatePickerModal({
  overlayId: 'mi-fecha-overlay',
  mode: 'single', // o 'range'
  title: 'Seleccionar fecha',
  maxDate: new Date(2029, 11, 31),
  onApply: function (result) {
    // single: result.date
    // range: result.startDate, result.endDate
    console.log(result);
  }
});
picker.open();

API devuelta

open(openOptions?) — muestra el modal; opcionalmente startDate, endDate, selectedDate, initialMonth.

close() / cancel() — oculta; cancel ejecuta onCancel.

setTitle(text) / getState() / destroy()
