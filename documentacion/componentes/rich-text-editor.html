Rich text editor

Editor de texto enriquecido con barra de formato (negrita, cursiva, subrayado, listas, alineación, imagen, enlace y más). Implementación en HTML, CSS y JavaScript puro: área contenteditable y comandos del navegador. Referencia: Creator v3 · Descripción.

Añadir editor enriquecido

En la pantalla [nombre], coloca el editor de texto enriquecido UBITS (rich-text-editor.css, button.css, fontawesome-icons.css, rich-text-editor.js). Incluye etiqueta, barra con negrita, listas, alineación, imagen y enlace, y área editable. Inicializa con initRichTextEditor o initAllRichTextEditors. Implementación

El proyecto es HTML, CSS y JS sin build: el área editable es un div con contenteditable="true". La barra usa document.execCommand (soportado en navegadores para este patrón; obsoleto a nivel estándar, suficiente para plantilla y prototipos).

El texto (Obligatorio) en .ubits-rich-text-editor__hint está oculto por defecto. Para mostrarlo, añade la clase ubits-rich-text-editor--show-required-hint al contenedor raíz del editor (desde la página o un control de preview); el componente JS no incluye interruptor interno para esto.
