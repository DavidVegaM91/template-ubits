Chip

Elemento compacto e interactivo para representar un filtro, selección o ítem removible. Suele usarse con icono, texto y opcionalmente botón de cierre.

Preview del componente

Los seis estados del Chip previsualizados a la vez. Elige el tamaño para ver cómo se aplica a todos.

Tamaño:

Hover Hover
Pressed Pressed

Características generales

Estados
Default, Hover, Pressed, Active, Focus, Disabled
Icono
Opcional a la izquierda (FontAwesome)
Cierre
Botón opcional para quitar el chip
Tamaños
XS (20px / 11px), SM (28px / 13px), MD (36px / 16px)

Prompts útiles

Copia el prompt, reemplaza lo que va entre [corchetes] y pide a Cursor que lo aplique.

Insertar Chip

Inserta un Chip en [ubicación] con texto "[texto]".

Chip con icono y botón quitar

Inserta un Chip en [ubicación] con icono [nombre del icono FontAwesome] a la izquierda, texto "[texto]" y botón para quitar. Usa clases ubits-chip ubits-chip--icon-left ubits-chip--close.

Documentación técnica

Información técnica para implementar el componente Chip.

Mostrar

Estructura del componente 

<!-- Chip solo texto -->
<span class="ubits-chip"><span class="ubits-chip__text">Chip</span></span>

<!-- Chip con icono izquierda y botón quitar -->
<span class="ubits-chip ubits-chip--icon-left ubits-chip--close">
  <i class="far fa-file-lines"></i>
  <span class="ubits-chip__text">nombre.pdf</span>
  <button type="button" class="ubits-chip__close" aria-label="Quitar"><i class="far fa-times"></i></button>
</span>

<!-- Chip activo (seleccionado) -->
<span class="ubits-chip ubits-chip--active ubits-chip--icon-left">
  <i class="far fa-file-lines"></i>
  <span class="ubits-chip__text">Activo</span>
</span>

<!-- Chip deshabilitado -->
<span class="ubits-chip ubits-chip--disabled ubits-chip--icon-left ubits-chip--close">
  <i class="far fa-file-lines"></i>
  <span class="ubits-chip__text">Disabled</span>
  <button type="button" class="ubits-chip__close" aria-label="Quitar" disabled><i class="far fa-times"></i></button>
</span>

Clases disponibles 

.ubits-chip                    /* Base (xs por defecto) */
.ubits-chip--xs                /* XS: 20px alto, 11px texto/iconos */
.ubits-chip--sm                /* SM: 28px alto, 13px texto/iconos */
.ubits-chip--md                /* MD: 36px alto, 16px texto/iconos */
.ubits-chip__text              /* Texto del chip */
.ubits-chip__close             /* Botón quitar (dentro del chip) */
.ubits-chip--icon-left         /* Icono a la izquierda */
.ubits-chip--close             /* Incluye botón de cierre */
.ubits-chip--active            /* Estado seleccionado */
.ubits-chip--disabled          /* Estado deshabilitado */

Cómo implementar

CSS: components/chip.css
Requisitos: ubits-colors.css, fontawesome-icons.css (si usas iconos)
JavaScript: No requerido. Para quitar chips, escucha clic en .ubits-chip__close. 
<link rel="stylesheet" href="components/chip.css">
<link rel="stylesheet" href="general-styles/fontawesome-icons.css">
