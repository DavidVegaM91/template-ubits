Status panel

Panel flotante en la esquina inferior para mostrar cargas, transferencias o tareas en curso. El usuario puede minimizarlo, cerrarlo y abrir cada ítem completado.

Preview del componente

El panel de ejemplo aparece dentro del recuadro. Usa los botones para añadir, completar o marcar error en las operaciones.

Características generales

Varias operaciones

Lista con scroll cuando hay muchos ítems.

Estados claros

En curso, completado, error o neutro.

Minimizar y cerrar

Botones oficiales UBITS en la cabecera.

Clic en ítem listo

Callback al completar para navegar o abrir detalle.

Prompts útiles

Insertar panel de actividad

Inserta el status panel UBITS en [ubicación] de la página. Importa status-panel.css, button.css y status-panel.js. Monta el HTML con renderStatusPanelHtml e initStatusPanel. Título: [texto del título].

Añadir operación en curso

Cuando el usuario inicie [acción], llama statusPanelAddItem con id único, título [texto] y estado loading. Al terminar, statusPanelUpdateItem con status success y subtítulo [texto].

Documentación técnica

Mostrar

Requisitos

  • general-styles/ubits-colors.css, ubits-typography.css, fontawesome-icons.css
  • components/button.css (acciones de cabecera)
  • components/status-panel.css + components/status-panel.js
  • Opcional icono IA: tokens --modo-ia-gradient-* en ubits-colors.css y useIaTitleIcon: true

API

renderStatusPanelHtml({ id, title, placement, titleIcon, useIaTitleIcon, hidden })
initStatusPanel(rootId, { title, autoTitle, titleLoading, titleDone, hidden })
statusPanelAddItem(root, itemId, { title, subtitle, status, icon, onClick })
statusPanelUpdateItem(root, itemId, { title, subtitle, status, icon, onClick })
statusPanelRemoveItem(root, itemId)
statusPanelClear(root)
statusPanelSetTitle(root, title)
statusPanelRefresh(root)

// status: 'loading' | 'success' | 'error' | 'neutral'
// placement: 'bottom-left' | 'bottom-right'
