File Upload Compact

Variante en línea del File Upload para espacios estrechos (modales, formularios): encabezado con título «Importar archivo» y botones de descarga de plantilla, tile vacío con borde dashed o tile cargado con icono o miniatura, hint de formatos, botones Subir/Cambiar y quitar con caneca. Incluye validación de tipo y peso, barra de progreso al cargar y estado de procesamiento como el componente base.

Características generales

Encabezado y plantillas
Título «Importar archivo» y hasta 3 botones de descarga, igual que el File Upload estándar.
Tile en línea
Ocupa una sola fila: hint de formatos, botón Subir y, al cargar, nombre + Cambiar + caneca.
Validación automática
Valida tipo y peso al seleccionar; borde rojo en tile vacío y mensaje debajo.
Miniatura opcional
Para imágenes muestra preview 40×40; otros tipos usan icono con fondo bg-2 y borde.
Informe de errores
Botón error-secondary junto al helper cuando el servidor detecta problemas en el contenido.
Quitar archivo
Mismo botón caneca error-tertiary que la card del File Upload estándar.
Modo oscuro
Todos los colores usan tokens UBITS, compatible con tema claro y oscuro automáticamente.

Prompts útiles

Usa estos prompts con Cursor AI para insertar y configurar el componente File Upload Compact.

Insertar tile compacto para logo o adjunto 

Añade el componente File Upload Compact UBITS en [ubicación]. Configúralo con:
- Título: "Importar archivo"
- 1 botón de descarga: "Descargar plantilla" (icono arrow-down-to-line)
- Tipos aceptados: CSV
- Límite: 5 MB
- Botones tile: Subir / Cambiar
Importa components/file-upload-compact.css, components/button.css, fontawesome-icons.css y components/file-upload-compact.js. Reemplaza los textos entre [corchetes].

Mostrar botón de informe de errores 

Cuando el servidor devuelva errores en el contenido del archivo en [ubicación], llama a fileUploadCompactShowErrorReport('[id-del-componente]', true, { placement: 'header' }) para mostrar el botón "Informe de errores" en el encabezado. Para errores tras procesar parcialmente, usa { placement: 'inline' } junto al mensaje debajo del tile.

CSV compacto en modal 

Añade File Upload Compact en [ubicación] del modal para subir CSV:
- accept: .csv
- maxSizeMb: 5
- formats: CSV · Hasta 5 MB
- icon: file-lines
- previewThumbnail: false
Importa file-upload-compact.css y file-upload-compact.js.

Mostrar un error externo (del servidor) 

Cuando la API devuelva un error al procesar el archivo en [ubicación], llama a fileUploadCompactSetError('[id-del-componente]', '[mensaje de error]'). Cuando el usuario quite el archivo, llama a fileUploadCompactClearError('[id-del-componente]').

Implementación con createFileUploadCompact() 

createFileUploadCompact({
  containerId: 'mi-contenedor',
  id: 'mi-file-upload-compact',
  title: 'Importar archivo',
  accept: '.csv,text/csv',
  maxSizeMb: 5,
  formats: 'CSV \u2022 Hasta 5 MB',
  icon: 'file-lines',
  uploadButtonLabel: 'Subir',
  changeButtonLabel: 'Cambiar',
  previewThumbnail: true,
  downloadButtons: [
    {
      label: 'Descargar plantilla',
      icon: 'arrow-down-to-line',
      onClick: function () { /* descargar CSV plantilla */ }
    }
  ],
  onChange: function (file, detail) {
    // file = File | null; detail.previewUrl = data URL si es imagen
  },
  onError: function (err) {
    // err = { type: 'type' | 'size', message: string }
  }
});

// Ocultar encabezado (p. ej. logo con etiqueta propia de la página)
fileUploadCompactSetHeaderVisible('mi-file-upload-compact', false);

// ── API auxiliar ──────────────────────────────────────────────
fileUploadCompactShowErrorReport('mi-file-upload-compact', true, { placement: 'header' });
fileUploadCompactSetError('mi-file-upload-compact', 'Se encontraron 3 filas inválidas.');
fileUploadCompactClearError('mi-file-upload-compact');
fileUploadCompactSetSuccess('mi-file-upload-compact', 'Archivo validado.');
fileUploadCompactSetProgress('mi-file-upload-compact', 45);
fileUploadCompactClearProgress('mi-file-upload-compact');
fileUploadCompactSetProcessingError('mi-file-upload-compact', { processedOk: 120, failed: 5 });
