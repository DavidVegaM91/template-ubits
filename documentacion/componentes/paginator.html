Paginator

Componente de paginación con navegación entre páginas y selector de items por página. Incluye posicionamiento inteligente del dropdown según el espacio disponible en el viewport.

Características generales

Navegación
Primera, anterior, números de página, siguiente y última; con una sola página se muestran igual, deshabilitadas, para la misma presencia visual en toda la app
Selector
Dropdown para cambiar items por página
Inteligente
Posicionamiento automático del dropdown

Documentación técnica

Información técnica detallada para implementar el componente Paginator.

Mostrar

Estructura del componente 

<!-- Container para el paginador -->
<div id="mi-paginador"></div>

<!-- JavaScript -->
<script>
loadPaginator('mi-paginador', {
    totalItems: 120,
    itemsPerPage: 16,
    currentPage: 1
});
</script>

Cómo implementar

🔧 REQUISITOS OBLIGATORIOS:

CSS Requerido:
• components/paginator.css - Estilos del componente
• components/button.css - Botones del paginador
• components/input.css - Selector de items por página
• fontawesome-icons.css - Iconos FontAwesome
• ubits-colors.css - Tokens de color UBITS
• ubits-typography.css - Tipografía UBITS

JavaScript Requerido:
• components/paginator.js - Lógica del componente
• components/input.js - Selector de items por página

Renderizado: JavaScript con función loadPaginator()

⚠️ IMPORTANTE: El paginador siempre debe estar centrado

El componente paginator siempre debe aparecer centrado en la página. El CSS del componente aplica automáticamente el centrado, pero si necesitas mostrar/ocultar el paginador con JavaScript, asegúrate de usar display: flex en lugar de display: block.

✅ CORRECTO: 
// Mostrar paginador (centrado)
const paginatorContainer = document.getElementById('paginador-container');
if (paginatorContainer) {
    paginatorContainer.style.display = 'flex'; // ✅ Usar flex para centrar
}
❌ INCORRECTO: 
// Mostrar paginador (NO centrado)
const paginatorContainer = document.getElementById('paginador-container');
if (paginatorContainer) {
    paginatorContainer.style.display = 'block'; // ❌ NO usar block, no permite centrar
}

El CSS del componente aplica automáticamente display: flex, justify-content: center y align-items: center a los contenedores comunes del paginador (como .paginador-container, .paginator-container, o IDs que contengan "paginador" o "paginator"). 

<!-- IMPLEMENTACIÓN BÁSICA -->
<!-- HTML -->

loadPaginator('mi-paginador', {

<script>
const paginator = loadPaginator('mi-paginador', {

    onPageChange: function(page) {
        console.log('Página cambiada a:', page);

    },
    onItemsPerPageChange: function(itemsPerPage) {
        console.log('Items por página cambiados a:', itemsPerPage);
        // Recargar contenido aquí

// Usar métodos públicos

API y funciones

loadPaginator(containerId, options)

Función principal para crear y renderizar el paginador.

Parámetros:
• containerId (string, requerido) - ID del contenedor donde se renderizará
• options.totalItems (number, requerido) - Número total de items a paginar
• options.itemsPerPage (number, default: 16) - Items por página por defecto
• options.currentPage (number, default: 1) - Página inicial
• options.itemsPerPageOptions (array, default: [16, 20, 32, 48]) - Opciones del selector
• options.onPageChange (function) - Callback cuando cambia la página
• options.onItemsPerPageChange (function) - Callback cuando cambia items por página
• options.showItemsSelector (boolean, default: true) - Mostrar selector

Retorna: Objeto con métodos públicos

Ejemplo: loadPaginator('mi-paginador', { totalItems: 120 })

Métodos públicos

goToPage(page) - Ir a una página específica
setItemsPerPage(itemsPerPage) - Cambiar items por página
getCurrentPage() - Obtener página actual
getItemsPerPage() - Obtener items por página actual
updateTotalItems(newTotalItems) - Actualizar total de items

Ejemplo:
const paginator = loadPaginator('mi-paginador', { totalItems: 120 });
paginator.goToPage(3);

Personalización

Cambiar opciones del selector

Personaliza las opciones disponibles en el selector de items por página: 

loadPaginator('mi-paginador', {
    totalItems: 120,
    itemsPerPage: 10,
    itemsPerPageOptions: [10, 20, 50, 100] // Opciones personalizadas
});

Integrar con contenido dinámico

Ejemplo de integración con contenido que se actualiza según la página: 

// Datos de ejemplo
const allItems = [/* ... 120 items ... */];
let currentPage = 1;
let itemsPerPage = 16;

// Función para renderizar items de la página actual
function renderItems() {
    const startIndex = (currentPage - 1) * itemsPerPage;
    const endIndex = startIndex + itemsPerPage;
    const pageItems = allItems.slice(startIndex, endIndex);
    
    // Renderizar items aquí
    console.log('Mostrando items:', pageItems);
}

// Crear paginador con callbacks
loadPaginator('mi-paginador', {
    totalItems: allItems.length,
    itemsPerPage: itemsPerPage,
    currentPage: currentPage,
    onPageChange: function(page) {
        currentPage = page;
        renderItems();
    },
    onItemsPerPageChange: function(newItemsPerPage) {
        itemsPerPage = newItemsPerPage;
        currentPage = 1;
        renderItems();
    }
});

// Renderizar inicialmente
renderItems();
