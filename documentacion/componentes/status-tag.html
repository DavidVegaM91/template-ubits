Status Tag

Etiqueta corta para mostrar un estado (activo, pendiente, error, etc.). Tiene varios colores y tamaños y puede llevar icono.

Preview del componente

Aquí puedes ver el componente Status Tag en acción. Usa los controles para cambiar variantes, tamaños y posiciones de icono.

Prompts útiles

Copia el prompt, reemplaza lo que va entre [corchetes] y pide a Cursor que lo aplique. Asegúrate de que importe status-tag.css y alert.css para que no se vea roto.

Insertar Status Tag

Inserta un Status Tag en [ubicación] con texto "[texto]" y color [success, info, warning, error o neutral] en tamaño [xs, sm, md o lg]. Debe importar status-tag.css y alert.css.

Status Tag con icono

Añade el icono [nombre del icono] al Status Tag de [ubicación], a la [izquierda o derecha]. Debe importar status-tag.css y alert.css.

Cambiar variante del status tag 

Cambia el status tag [descripción_del_tag] de variante [variante_actual] a [nueva_variante: success, info, warning, error o neutral].

Cambiar tamaño del status tag 

Cambia el tamaño del status tag [descripción_del_tag] a [xs, sm, md o lg].

Cambiar texto del status tag 

Cambia el texto del status tag [descripción_del_tag] para que diga "[nuevo_texto]".

Quitar icono del status tag 

Quita el icono del status tag [descripción_del_tag]. Elimina la clase ubits-status-tag--icon-left o ubits-status-tag--icon-right y el elemento <i> del icono.
