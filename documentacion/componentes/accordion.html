Accordion

Componente para mostrar y ocultar bloques de información de forma ordenada, con título obligatorio y campos opcionales de numeración y descripción.

Preview del componente

Cambia el tamaño, la variante y activa o desactiva numeración y descripción (debajo del título) para ver las variaciones del Accordion.

Numeración
Descripción

Características generales

Cuatro tamaños
SM, MD y LG para diferentes jerarquías de texto
Título obligatorio
Siempre muestra el tema principal del bloque
Campos opcionales
La descripción opcional va debajo del título en el encabezado
Dos variantes
Con fondo completo o solo borde inferior sin fondo

Prompts útiles

Reemplaza los textos entre [corchetes] antes de enviarlos.

Insertar accordion básico 
Inserta un Accordion UBITS en [ubicación] en tamaño [sm/md/lg], variante [filled/line], con título [texto]. Importa accordion.css y accordion.js.
Accordion con numeración 
Inserta un Accordion UBITS en [ubicación] en tamaño [sm/md/lg], variante [filled/line], con numeración [ej: 1], título [texto] y descripción debajo del título [texto].
Actualizar tamaño y contenido 
Cambia el Accordion [nombre o selector] a tamaño [sm/md/lg] y variante [filled/line], mantén el título [texto], ajusta la descripción debajo del título a [texto] y deja el contenido interno con [HTML libre].

<div class="ubits-accordion ubits-accordion--md ubits-accordion--filled" data-open="false">
  <button type="button" class="ubits-accordion__header" aria-expanded="false" aria-controls="faq-content">

      <span class="ubits-accordion__number ubits-body-md-regular">1</span>

  </button>

    <!-- Aquí puede ir cualquier HTML: tablas, texto, formularios, etc. -->

  </div>
</div>
Cómo implementar 
<link rel="stylesheet" href="../../components/accordion.css">
<script src="../../components/accordion.js"></script>

<div id="faq-accordion"></div>

<script>
createAccordion({
  containerId: 'faq-accordion',
  size: 'md',
  variant: 'filled', // 'filled' | 'line'
  title: '¿Cómo funciona este proceso?',
  number: '1',
  description: 'Descripción opcional debajo del título.',
  contentHtml: '<p class="ubits-body-md-regular">Contenido libre dentro del accordion.</p>',
  isOpen: false
});
</script>

API

createAccordion(config): renderiza un accordion en un contenedor.
Config: containerId (obligatorio), title (obligatorio), size (sm|md|lg), variant (filled|line), number (opcional), description (opcional, se muestra debajo del título), contentHtml (opcional, contenido interno libre), isOpen (opcional).
initAccordions(selector): inicializa acordeones ya renderizados en HTML.
toggleAccordion(target): abre o cierra un accordion por selector o elemento.
