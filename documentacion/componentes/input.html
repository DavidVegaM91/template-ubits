Características generales

El componente Input de UBITS es un sistema completo de campos de entrada que cubre todos los casos de uso comunes en aplicaciones web modernas.

11 Tipos de Input
text, email, password, number, tel, url, select, textarea, search, autocomplete, calendar
4 Tamaños
XS (28px), Small (32px), Medium (40px), Large (48px)
5 Estados
default, hover, focus, invalid, disabled
Iconos FontAwesome
Soporte completo para iconos izquierdo y derecho
Contador de Caracteres
Independiente del helper text, con límite configurable
Helper Text
Texto de ayuda opcional, independiente del contador
Mandatory/Optional
Texto indicativo de campo obligatorio u opcional
Validación Manual
Sistema de validación manual para email, tel, url
Scroll Infinito
Carga automática en SELECT con 50+ opciones
Botón Limpiar
Botón X en SEARCH y AUTOCOMPLETE que aparece al escribir
Toggle Password
Mostrar/ocultar contraseña con icono de ojo
Date Picker
Navegación por mes y año para fechas antiguas

Prompts útiles

Usa estos prompts con Cursor AI para configurar y personalizar el componente Input de manera eficiente.

Crear input básico 

Crea un input de texto que diga "[tu_label]" con placeholder "[tu_placeholder]" en [ubicación donde quieres el input].

Crear input de email con validación 

Crea un input de email que diga "[tu_label]" con placeholder "[tu_placeholder]" y validación manual en [ubicación donde quieres el input].

Crear input de contraseña con toggle 

Crea un input de contraseña que diga "[tu_label]" con placeholder "[tu_placeholder]" y botón para mostrar/ocultar en [ubicación donde quieres el input].

Crear select con opciones 

Crea un select que diga "[tu_label]" con placeholder "[tu_placeholder]" y las opciones [lista_de_opciones] en [ubicación donde quieres el input].

Crear textarea multilínea 

Crea un textarea que diga "[tu_label]" con placeholder "[tu_placeholder]" en [ubicación donde quieres el input].

Crear input de búsqueda con limpiar 

Crea un input de búsqueda que diga "[tu_label]" con placeholder "[tu_placeholder]" y botón X para limpiar en [ubicación donde quieres el input].

Crear autocomplete con sugerencias 

Crea un autocomplete que diga "[tu_label]" con placeholder "[tu_placeholder]" y las sugerencias [lista_de_sugerencias] en [ubicación donde quieres el input].

Crear input de fecha con calendario 

Crea un input de fecha que diga "[tu_label]" con placeholder "[tu_placeholder]" y date picker en [ubicación donde quieres el input].

Agregar iconos al input 

Agrega el icono [nombre_del_icono] [izquierdo/derecho] al input [descripción_del_input].

Agregar contador de caracteres 

Agrega contador de caracteres con límite de [número] al input [descripción_del_input].

Agregar helper text 

Agrega helper text que diga "[tu_mensaje_de_ayuda]" al input [descripción_del_input].

Marcar input como obligatorio 

Marca el input [descripción_del_input] como obligatorio con texto "(obligatorio)".

Cambiar tamaño del input 

Cambia el tamaño del input [descripción_del_input] a [xs, small, medium o large].

Cambiar estado del input 

Cambia el estado del input [descripción_del_input] a [default, hover, focus, invalid o disabled].

Crear formulario completo 

Crea un formulario de [nombre_del_formulario] con los inputs: [lista_de_inputs_necesarios] en [ubicación donde quieres el formulario].
