Empty State

Mensaje que se muestra cuando no hay contenido: icono, título, descripción y hasta dos botones. Sirve para listas vacías, búsquedas sin resultados o pantallas iniciales.

Características generales

Icono
Icono grande arriba; puedes elegir tamaño del contenedor
Título y descripción
Textos que explican por qué está vacío y qué hacer
Dos botones opcionales
Uno secundario y uno principal para acciones (ej. Crear, Buscar)
Responsive
Se adapta a diferentes tamaños de pantalla
Ancho completo
Puede ocupar todo el ancho en secciones con flex column

Prompts útiles

Copia el prompt, reemplaza lo que va entre [corchetes] y pide a Cursor que lo aplique. Asegúrate de que importe empty-state.css, button.css y alert.css para que no se vea roto.

Insertar Empty State

Inserta un Empty State en [ubicación] cuando no haya [contenido / resultados / etc.]. Título "[título]", descripción "[descripción]" e icono [nombre del icono]. Debe importar empty-state.css, button.css y alert.css.

Añadir botones al Empty State

En el Empty State de [ubicación], añade dos botones: uno que diga "[texto del botón secundario]" y otro "[texto del botón principal]". Debe importar button.css y alert.css.

Cambiar título y descripción

Cambia el empty state de [nombre del contenedor]: que el título sea "[nuevo título]" y la descripción "[nueva descripción]".

Empty state que ocupe todo el ancho

Haz que el empty state de [nombre del contenedor] ocupe todo el ancho de la sección. Coloca el contenedor directamente en la sección (section-single) con width: 100% y asegúrate de que la sección tenga flex-direction: column.
