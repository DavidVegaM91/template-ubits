Notifications menu

Panel de notificaciones del header: campana con puntito de atención y lista de avisos (título, descripción y fecha). No es una variante del dropdown-menu.

Vista previa

Uso

createNotificationsMenu({
  items: [
    {
      id: '1',
      title: 'Evaluaciones vencidas',
      description: '27 colaboradores no han completado su evaluación.',
      date: 'Hoy, 9:12 a.m.',
      onClick: function () {}
    }
  ],
  hasUnread: true
})

Importa notifications-menu.css, notifications-menu.js, button.css y attention-badge.css.
