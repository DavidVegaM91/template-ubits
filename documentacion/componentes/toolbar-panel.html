Toolbar panel

Bloque con fondo para la cabecera de una lista o tabla: título, texto de apoyo (por ejemplo recuento) y acciones alineadas a la derecha. Sirve en catálogos, inventarios, LMS Creator y cualquier pantalla que necesite la misma barra de contexto.

Características generales

Contenedor único
Fondo y radios coherentes con el resto de tarjetas del producto
Barra flexible
Título y acciones se reparten; en móvil la barra se apila de forma ordenada
Misma pieza en varios flujos
Catálogos, LMS Creator u otras listas pueden compartir la misma barra; lo que cambia es qué acciones muestras
Componible
Los botones son el componente Button oficial; puedes añadir o quitar grupos según la pantalla

Barra de lista reutilizable

En [ubicación de la página], añade un bloque toolbar panel (título [título], meta con contador y acciones). Sigue la sección «Toolbar panel» del README y las reglas Cursor «Implementar Toolbar panel»: importa toolbar-panel.css, button.css, fontawesome y el resto de dependencias según si hay búsqueda, select, modal, chips o empty state.

Búsqueda expandible en la barra

En [pantalla], cablea la barra como en Contenidos: lupa que oculta el botón y muestra createInput type search en .ubits-toolbar-panel__search-inline; clic fuera sin texto colapsa. Si hay filtros con select: dropdown-menu.css y dropdown-menu.js antes de input.js. Actualiza el contador del meta con visibles/total y aplica el patrón del README para el botón Filtrar (active + badge).

Clases principales

  • ubits-toolbar-panel: contenedor del bloque (fondo, padding, apilado vertical).
  • ubits-toolbar-panel__bar: fila principal título versus acciones.
  • ubits-toolbar-panel__title-block, __title, __meta: cabecera izquierda.
  • ubits-toolbar-panel__actions: zona derecha (búsqueda, filtros, conmutadores).
  • ubits-toolbar-panel__search-group, __search-inline, __search-toggle, __filter-btn: patrón lupa + input + filtro.
  • ubits-toolbar-panel__control-group, __control-label: grupo de botones de vista u opciones secundarias.

Referencia en producto: ubits-colaborador/lms-creator/contenidos.html (lista de contenidos del LMS Creator).

Checklist de imports y piezas relacionadas

  1. Base + button.css + toolbar-panel.css.
  2. Iconos en barra: tooltip.css + tooltip.js + initTooltip tras cambios dinámicos.
  3. Búsqueda expandible: input.css + input.js + contenedor .ubits-toolbar-panel__search-inline.
  4. Select en filtros: dropdown-menu.css + dropdown-menu.js antes de input.js.
  5. Modal de filtros: modal.css + modal.js.
  6. Chips / empty / tabla: chip.css, empty-state.css, table.css según aplique.
  7. Patrones de producto: README «Un solo botón Filtrar» y «Empty state cuando no hay resultados».
