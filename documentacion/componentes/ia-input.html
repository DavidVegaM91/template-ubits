IA Input

Input oficial para flujos IA: variantes panel (modales LMS, IA panel) y chat (Modo Estudio). API: createUbitsIAInput(options) — paridad con UbitsIAInput en React.

Características generales

Variantes
Panel (modales) y Chat (Modo Estudio)
Contador
Palabras o caracteres con mín/máx y separador de miles
Adjuntos
Imágenes en miniatura y chips de archivos sobre el prompt
Acciones
IA-Button con tokens, Enviar o sin botón (SCORM)

Uso básico 

<link rel="stylesheet" href="components/ia-input.css">
<script src="components/ia-input.js"></script>

<div id="mi-ia-input"></div>
<script>
  var iaInput = createUbitsIAInput({
    variant: 'panel',
    placeholder: 'Describe tu portada de forma específica',
    attach: true,
    counter: { min: 6, max: 500, unit: 'palabras' },
    action: { type: 'ia', label: 'Generar portada', tokenCost: 2, onClick: function () {} },
    onChange: function (v) { /* … */ }
  }).mount('mi-ia-input');
</script>
