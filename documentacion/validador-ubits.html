Validaciones automáticas

Tokens de color

  • Detecta colores hardcodeados (hex, rgb, rgba, etc.)
  • Verifica uso de tokens UBITS
  • Comprueba importación de ubits-colors.css

Tipografía UBITS

  • Detecta clases inventadas (ubits-h1, ubits-title, etc.)
  • Verifica clases oficiales UBITS
  • Comprueba importación de ubits-typography.css

Tokens de espaciado y estilos

  • Detecta font-size en px → usar var(--font-size-xs|sm|md|h2|h1|d4|d3|d2|d1)
  • Detecta border-radius en px → usar var(--border-radius-xs|sm|md|lg|xl|full)
  • Detecta gap en px → usar var(--gap-xs|sm|md|lg|xl|2xl|4xl)
  • Detecta padding en px → usar var(--padding-xs|sm|md|lg|xl|2xl|4xl|6xl)
  • Comprueba importación de styles.css / ubits-spacing-tokens.css

Componentes UBITS (30+)

  • Navegación: SubNav, Sidebar, TabBar, Floating Menu, Profile Menu
  • UI básicos: Button, IA-Button, Alert, Toast, Input (11 tipos)
  • Contenido: Card Content, Card Content Compact, Carousel
  • Avanzados: Header Product, Status Tag, Tab, Empty State, Paginator
  • Chat/IA: Copilot Chat, Study Chat
  • Utilidad: Tooltip
  • Valida CSS/JS requeridos para cada componente
