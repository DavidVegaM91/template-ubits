Componentes UBITS

Una colección completa de componentes UBITS listos para usar. Botones, cards, formularios y más, todos con la identidad visual oficial.

Componentes pre-construidos

Botones, cards, formularios, navegación y más. Todos los componentes con estilos UBITS aplicados y listos para copiar y pegar.

Fácil personalización

Modifica colores, tamaños y comportamientos con variables CSS. Sin tocar el código base, solo ajusta los tokens.

Consistencia garantizada

Todos los componentes siguen los mismos principios de diseño. Espaciado uniforme y tipografía consistente. Modo claro y oscuro automático.

¿Cómo usar los componentes?

1

Selecciona el componente

Navega por la biblioteca y encuentra el componente que necesitas para tu interfaz.

2

Copia el código

Copia el HTML y CSS del componente directamente a tu proyecto. Sin instalaciones ni configuraciones.

3

Personaliza y usa

Ajusta los textos, colores y comportamientos según tus necesidades específicas.
